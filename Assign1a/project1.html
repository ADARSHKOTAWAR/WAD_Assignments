Apple Freshness Detection Project

Description: This project focuses on detecting, counting, and analyzing the freshness of apples using deep learning techniques. It employs YOLO for object detection and models like VGG16 and ResNet50 for freshness analysis.

Duration: 3 months

Role: Full-stack Developer

Technologies Used:

  • Python
  • YOLO
  • OpenCV
  • Django
  • HTML
  • CSS
  • Bootstrap

Features:

  • Detect apples in images and videos with high accuracy.
  • Analyze the freshness of apples based on visual features.
  • Interactive UI for users to upload images.

Challenges Faced:

  • Optimizing YOLO for real-time performance.
  • Training models with limited labeled datasets.
