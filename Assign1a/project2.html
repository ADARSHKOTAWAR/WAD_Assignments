DSA Tracker

Description: A platform developed for students to track their progress in Data Structures and Algorithms (DSA) through quizzes and programming challenges. The system helps users improve their skills by offering real-time feedback and performance analysis.

Duration: 3 months

Role: Full-stack Developer

Technologies Used:

  • HTML
  • CSS
  • Node.js
  • Thender Client
  • CodeMirror
  • Compliex (NPM)

Features:

  • Interactive quizzes to test DSA knowledge.
  • Real-time coding environment with code editor and compiler integration.
  • Progress tracking to visualize improvement over time.

Challenges Faced:

  • Ensuring smooth integration of real-time coding features and quiz systems.
  • Optimizing the performance of the platform with a large number of users.
